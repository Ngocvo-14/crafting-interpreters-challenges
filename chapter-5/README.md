Chapter 5: Representing Code - Challenges
Challenge 3: RPN Converter - Convert expressions to Reverse Polish Notation (RPN).

What is RPN?
In RPN, operators come after their operands:
Infix: `1 + 2` → RPN: `1 2 +`
Infix: `(1 + 2) \* (4 - 3)` → RPN: `1 2 + 4 3 - \*`

Running the Code

Compile (make sure you have Expr.java, Token.java, TokenType.java)
